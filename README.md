# 🚀 yottadb silly server (yss)
100% MUMPS 0 middleware 0 bloat 100% HONK
## 📜 the seven core tenets
1. always obscure ansi mumps there is no `$DATA` here only `$D`; single letters only, fear the abbreviation
2. make it ydb specific exploit raw `ZLISTEN` parameters and background `JOB` commands, no polyfills nor portability
3. make it fast worker processes branch in microseconds since headers are flushed instantly
4. make it silly ecause enterprise software deserves silly too sometimes
5. make it stupid why use a JSON library when you can manually concatenate strings using deeply nested GOTO loops?
6. make it beautiful ruthless alignment, every space has a purpose
7. make it autistic do not care about `User-Agents` but care about exact byte boundaries `$C(13,10)` and absolute routing rigidity
## 🛠️ installation and usage
1. [install YottaDB](https://download.yottadb.com/ydbinstall.sh) and ensure UTF-8 mode is active (`export ydb_chset=utf-8`)
2. clone this repository into your YottaDB routines directory
3. open your terminal and enter the MUMPS prompt
   ```bash
   ydb
   ```
4. seed the silly database
   ```mumps
   YDB> D SEED^ZDB
   ```
5. run the Unholy Forge to dynamically generate your emoji routing monolith (`ZEMOJI.m`)
   ```mumps
   YDB> D ^ZFORGE
   ```
6. summon the Daemon on port 8080 and then ??? profit
   ```mumps
   YDB> D ^ZSRV
   ```
